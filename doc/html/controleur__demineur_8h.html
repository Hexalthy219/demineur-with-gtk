void click_decouvre_case ( GtkWidget *  pButton,
GdkEventButton *  type_click,
gpointer  data 
)

fonction callback du signal d'un des boutons du champ de mine

Parameters
dataun pointeur vers Data_for_Callback contenant les données nécessaires
type_clickun pointeur vers GdkEventButon, structure contenant un champ "button" : une valeur déterminant le type de clique effectué par le signal
pButtonun pointeur vers GtkWidget, le bouton ayant émis le signal
Precondition
data!=NULL

Lance la découverte récursive des cases à partir de la position du bouton ayant émis le signal si l'utilisateur à fait un clique gauche.
Lance la découverte récursive pour toutes les case, autour du bouton ayant émis le signal, n'ayant pas de drapeau si l'utilisateur a fait un clique molette et que le nombre de drapeau autour du bouton coincide avec le nombre de mine autour.
Affiche un drapeau sur la case ou le retire si déjà affiché, si l'utilisateur a fait un clique droit.
